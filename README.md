# AZ-AB-Reporting

A reporting platform for managing and visualizing multi-tenant data.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abdelaliii/AZ-AB-Reporting.git
   cd AZ-AB-Reporting
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   - Copy `.env.example` to `.env`
   - Update any necessary environment variables in `.env`

### Running the Application

To start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173` (or the port specified in your console).

## Running with Docker (Optional)

If you prefer using Docker, you can run the application with:

```bash
docker-compose up --build
```

The application will be accessible at `http://localhost:3000`.
